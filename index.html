📖 Phone থেকে Contacts Export করুন

Desktop Chrome, Firefox, Safari — সব browser-এ এই method কাজ করে।

🤖 Android
1
Contacts app খুলুন
2
Menu (⋮) → Manage Contacts → Export তে যান
3
.vcf format সিলেক্ট করে phone storage-এ save করুন
4
উপরের Upload VCF zone-এ সেই file টি দিন
🍎 iPhone (iOS)
1
iCloud.com অথবা Settings → Apple ID → iCloud → Contacts থেকে export করুন
2
iCloud.com → Contacts → সব select (Ctrl+A) → Export vCard
3
Download হওয়া .vcf file উপরে upload করুন
🌐 Google Contacts (Desktop)
1
contacts.google.com খুলুন
2
বাম পাশে Export → vCard (.vcf) সিলেক্ট করুন
3
Download হওয়া file উপরে Upload VCF-এ দিন
